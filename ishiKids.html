Ishihara Plates CVD Test (For kids < 6 years of age)

The Ishihara Test is the most well known color blindness test all around the world. This test has been modified so that you can determine if a child has red-green CVD. According to what the child can see, the test gives feedback of the degree of red-green color vision deficiency.

This test can only be used to detect red-green color blindness. This test is not very accurate since the colors of the printed books cannot be simulated by a display. So before coming to a conclusion, we urge you to finish all of the tests.

How the test works:

For each plate you have to select which animal you see in the plate.

If you don't see any animal, press NEXT!
